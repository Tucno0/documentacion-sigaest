Contribuir al desarrollo de **SIGAEST** es un proceso sencillo y accesible. Sin embargo, para garantizar la calidad y seguridad del sistema, solo pueden realizar cambios aquellos que cumplan ciertos requisitos.

## Quién puede contribuir

Para contribuir al desarrollo de **SIGAEST**, es necesario cumplir al menos una de las siguientes condiciones:

- Ser parte del **equipo de desarrolladores** del sistema.
- Pertenecer al **personal de una institución** que utiliza SIGAEST y contar con la autorización correspondiente.

Si cumples con estos requisitos, puedes solicitar acceso al repositorio y coordinar con el equipo de desarrollo para realizar mejoras o correcciones en el sistema.

## Pasos para contribuir

1. **Solicitar acceso**
   - Contactar con el equipo de desarrollo para obtener los permisos necesarios.
   
2. **Configurar el entorno**
   - Clonar el repositorio y seguir los pasos de instalación.

3. **Realizar cambios**
   - Seguir las normas de desarrollo establecidas.
   - Documentar adecuadamente las modificaciones.

4. **Solicitar revisión**
   - Enviar una solicitud para que los cambios sean revisados y aprobados por el equipo.

## Contacto
Si deseas contribuir o tienes preguntas sobre el proceso, puedes comunicarte con el equipo de desarrollo para obtener más información.
